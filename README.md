# 🎮 GameMarket — Маркетплейс цифровых товаров

Платформа для покупки/продажи игр, аккаунтов и ключей активации.  
**Технологии:** Next.js 14, Nest.js, Prisma, PostgreSQL, YooKassa, TailwindCSS.

## 🚀 Возможности MVP

- ✅ Просмотр каталога игр, ключей, аккаунтов
- ✅ Регистрация / вход (email + Google)
- ✅ Добавление товаров от пользователей
- ✅ Корзина и оформление заказа
- ✅ Оплата через **YooKassa** (карты, SBP, Apple Pay)
- ✅ Автоматическая выдача товара после оплаты
- ✅ Личный кабинет с историей покупок
- ✅ Адаптивный дизайн

## 🛠 Технологический стек

| Компонент | Технология |
|-----------|------------|
| Frontend | Next.js 14 (App Router) |
| Backend | Nest.js + Fastify |
| ORM | Prisma |
| Database | PostgreSQL |
| Payments | YooKassa |
| Auth | NextAuth.js |
| Styling | TailwindCSS + shadcn/ui |
| State | Zustand (корзина) |

## 📋 Требования

- Node.js 18+
- PostgreSQL 14+
- YooKassa аккаунт (shopId + secretKey)
